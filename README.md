# Heart Disease Prediction Project

## Objective
Predict heart disease using machine learning models.

## Dataset
UCI-based Heart Disease dataset.

## Steps
- Data preprocessing
- Feature engineering
- Model training (Logistic Regression, KNN, Random Forest)
- Evaluation

## Results
Random Forest achieved the best performance.

## Deployment
A Streamlit app was created to allow user input and prediction.
